Definations

website==> A website is a collection of interconnected web pages under a single domain, accessible via the internet. Websites serve various purposes, including providing information, services, entertainment, and e-commerce. They can be created using HTML, CSS, JavaScript, and other programming languages. Websites range from simple static pages to complex dynamic applications, incorporating multimedia elements like images, videos, and interactive features. They are hosted on servers and accessed through web browsers. Effective websites focus on user experience, ensuring easy navigation, fast loading times, and responsive design to accommodate different devices and screen sizes.

WWW:A system of interlinked hypertext documents accessed via the internet. Users can view web pages that may contain text, images, videos, and other multimedia and navigate between them using hyperlinks

HTTP (Hypertext Transfer Protocol):

Client-Server:A computing model where the client requests services and resources from the server. The server processes these requests and returns the requested information to the client.

Domain:The address where Internet users can access your website. It is the name of the website, like example.com.

Host:A computer or other device connected to a computer network. In web hosting, it refers to the service that runs websites and delivers them to users.

Hosting:The service of storing and serving website files on a server so that they are accessible on the internet. Web hosting providers offer services like storage, security, and support.

FTP(File Transfer Protocol):A standard network protocol used to transfer computer files between a client and server on a computer network. It is commonly used for uploading and downloading files from a server.

path:The part of a URL that specifies the location of a resource on the server. It follows the domain name and typically indicates a specific file or directory.

Extension:The suffix at the end of a filename, indicating the file type. For example, .html for HTML files, .jpg for JPEG images, and so on.

Editor/Vscode:A source-code editor developed by Microsoft. It supports development operations like debugging, task running, and version control. It is widely used by developers for coding in multiple languages.

URL/URI/URN:

URL (Uniform Resource Locator):The address of a resource on the internet. It consists of the protocol, domain, path, and optionally other components like port, query, and fragment.

URI (Uniform Resource Identifier):A string of characters used to identify a resource either by location, name, or both. URLs are a type of URI.

URN (Uniform Resource Name):A type of URI that identifies a resource by name in a particular namespace, but not by its location.

Address Bar: text field in a web browser where the URL is displayed and can be entered.

Title Bar: part of a web browser window or application window that displays the title of the web page or document that is currently open.
